# Portfolio Map

This file is the master inventory of Marshall Epie's products, services, and internal systems.

## Portfolio structure

### Umbrella
- **Marshall Epie Technologies**
  - Umbrella identity for products and services created by Marshall Epie
  - Previously operated as a limited company
  - Now functions as a lean umbrella rather than a formal limited-company operating shell

### Public hub
- **marshallepie.com**
  - Central marketing location
  - Represents software products, music, and electrical services
  - Routes attention toward the broader Marshall Epie portfolio

### Internal operating layer
- **MEPA**
  - Internal operations and continuity system
  - Current state: documentation-first repo
  - Intended future state: private app/dashboard for portfolio management and AI interaction
